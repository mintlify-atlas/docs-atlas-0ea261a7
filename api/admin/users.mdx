## List users

### GET /api/users

Retrieve a paginated list of all users with filtering and search.

#### Query parameters

<ParamField query="is_admin" type="boolean">
  Filter by admin status. `true` for admins only, `false` for customers only.
</ParamField>

<ParamField query="search" type="string">
  Search by name or email
</ParamField>

<ParamField query="sort_by" type="string">
  Sort field. Options: `name`, `email`, `created_at`, `updated_at`. Default: `created_at`.
</ParamField>

<ParamField query="sort_order" type="string">
  Sort direction. Options: `asc`, `desc`. Default: `desc`.
</ParamField>

<ParamField query="per_page" type="integer">
  Items per page. Range: 1-100. Default: 20.
</ParamField>

#### Response

```json
{
  "status": "success",
  "data": {
    "current_page": 1,
    "data": [
      {
        "id": "9d3c5e7f-1234-5678-9abc-def012345678",
        "name": "John Doe",
        "email": "john@example.com",
        "is_admin": false,
        "created_at": "2026-03-01T10:00:00.000000Z",
        "updated_at": "2026-03-01T10:00:00.000000Z"
      }
    ],
    "per_page": 20,
    "total": 150
  }
}
```

## Create user

### POST /api/users

Create a new user account.

#### Request body

<ParamField body="name" type="string" required>
  User's full name. Maximum 255 characters.
</ParamField>

<ParamField body="email" type="string" required>
  User's email address. Must be unique.
</ParamField>

<ParamField body="password" type="string" required>
  User password. Minimum 8 characters.
</ParamField>

<ParamField body="password_confirmation" type="string" required>
  Password confirmation. Must match password.
</ParamField>

<ParamField body="is_admin" type="boolean">
  Whether the user should have admin privileges. Default: `false`.
</ParamField>

#### Example request

```bash
curl -X POST https://api.grancamerino.com/api/users \
  -H "Authorization: Bearer ADMIN_JWT_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "Jane Smith",
    "email": "jane@example.com",
    "password": "securePassword123",
    "password_confirmation": "securePassword123",
    "is_admin": false
  }'
```

## Delete user

### DELETE /api/users/{user}

Delete a user account.

<Warning>You cannot delete your own account. Returns 403 error if attempted.</Warning>

#### Response

```json
{
  "status": "success",
  "message": "User deleted successfully"
}
```

## Toggle admin status

### PATCH /api/users/{user}/toggle-admin

<Warning>You cannot change your own admin status. Returns 403 error if attempted.</Warning>

#### Response
