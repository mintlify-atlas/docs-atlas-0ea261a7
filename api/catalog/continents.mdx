## Create Continent

<api method="POST" url="https://api.grancamerino.com/api/continents" auth="Bearer" />

<Note type="warning">
  **Admin Only** - Requires authentication with admin privileges
</Note>

Create a new continent.

### Headers

<ParamField header="Authorization" type="string" required>
  Bearer token with admin privileges
</ParamField>

### Body Parameters

<ParamField body="name" type="string" required>
  Continent name (max 255 characters)
</ParamField>

### Response

<ResponseField name="status" type="string">
  Response status ("success")
</ResponseField>

<ResponseField name="data" type="object">
  Created continent object
</ResponseField>

<RequestExample>
```bash cURL
curl -X POST https://api.grancamerino.com/api/continents \
  -H "Authorization: Bearer YOUR_ADMIN_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "Antarctica"
  }'
```
</RequestExample>

<ResponseExample>
```json Response
{
  "status": "success",
  "data": {
    "id": "9d445a8c-8c3a-4d9d-bb4c-f0a6c2e8b507",
    "name": "Antarctica",
    "created_at": "2024-03-03T15:00:00.000000Z",
    "updated_at": "2024-03-03T15:00:00.000000Z"
  }
}
```
</ResponseExample>

  Continent UUID
</ParamField>

### Body Parameters

<ParamField body="name" type="string" optional>
  Continent name (max 255 characters)
</ParamField>
