## List All Sizes

<api method="GET" url="https://api.grancamerino.com/api/sizes" />

Retrieve all available product sizes, ordered alphabetically by name.

### Response

<ResponseField name="status" type="string">
  Response status ("success")
</ResponseField>

<ResponseField name="data" type="array">
  Array of size objects, ordered by name
  
  <Expandable title="Size Object">
    <ResponseField name="id" type="string">
      UUID of the size
    </ResponseField>
    
    <ResponseField name="name" type="string">
      Size name (e.g., "S", "M", "L", "XL", "XXL")
    </ResponseField>
    
    <ResponseField name="created_at" type="string">
      ISO 8601 timestamp
    </ResponseField>
    
    <ResponseField name="updated_at" type="string">

</ResponseField>

## Create Size

<api method="POST" url="https://api.grancamerino.com/api/sizes" auth="Bearer" />

<Note type="warning">
  **Admin Only** - Requires authentication with admin privileges
</Note>

Create a new product size.

### Headers

<ParamField header="Authorization" type="string" required>
  Bearer token with admin privileges
</ParamField>

### Body Parameters

<ParamField body="name" type="string" required>
  Size name (max 255 characters, must be unique)
</ParamField>

### Response

<ResponseField name="status" type="string">
  Response status ("success")
</ResponseField>

<ResponseField name="data" type="object">
  Created size object
</ResponseField>

<RequestExample>
```bash cURL
curl -X POST https://api.grancamerino.com/api/sizes \
  -H "Authorization: Bearer YOUR_ADMIN_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "XXXL"
  }'
```
</RequestExample>

<ResponseExample>
```json Response
{
  "status": "success",
  "data": {
    "id": "9d445a8c-8c3a-4d9d-bb4c-f0a6c2e8b606",
    "name": "XXXL",
    "created_at": "2024-03-03T15:10:00.000000Z",
    "updated_at": "2024-03-03T15:10:00.000000Z"
  }
}
```
</ResponseExample>

## Update Size

<api method="PUT" url="https://api.grancamerino.com/api/sizes/{id}" auth="Bearer" />

<Note type="warning">
  **Admin Only** - Requires authentication with admin privileges
</Note>

Update an existing size.

### Headers

<ParamField header="Authorization" type="string" required>
  Bearer token with admin privileges
</ParamField>

### Path Parameters

<ParamField path="id" type="string" required>
  Size UUID
</ParamField>

### Body Parameters

<ParamField body="name" type="string" required>
  Size name (max 255 characters, must be unique except for current size)
</ParamField>

### Response

<ResponseField name="status" type="string">
  Response status ("success")
</ResponseField>

<ResponseField name="data" type="object">
  Updated size object
</ResponseField>

<RequestExample>
```bash cURL

  -H "Authorization: Bearer YOUR_ADMIN_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "3XL"
  }'
```
</RequestExample>

<ResponseExample>
