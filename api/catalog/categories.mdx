## List All Categories

<api method="GET" url="https://api.grancamerino.com/api/categories" />

Retrieve all product categories in the system.

### Response

<ResponseField name="status" type="string">
  Response status ("success")
</ResponseField>

<ResponseField name="data" type="array">
  Array of category objects
  
  <Expandable title="Category Object">
    <ResponseField name="id" type="string">
      UUID of the category
    </ResponseField>
    
    <ResponseField name="name" type="string">
      Category name (e.g., "Men", "Women", "Kids")
    </ResponseField>
    
    <ResponseField name="slug" type="string">
      URL-friendly slug for the category
    </ResponseField>
    
    <ResponseField name="description" type="string" optional>
      Category description
    </ResponseField>
    
    <ResponseField name="image_url" type="string" optional>
      URL to category image
    </ResponseField>
    
    <ResponseField name="is_active" type="boolean">
      Whether the category is active

      ISO 8601 timestamp

    <ResponseField name="updated_at" type="string">

## Create Category

<api method="POST" url="https://api.grancamerino.com/api/categories" auth="Bearer" />

<Note type="warning">
  **Admin Only** - Requires authentication with admin privileges
</Note>

Create a new product category.

### Headers

<ParamField header="Authorization" type="string" required>
  Bearer token with admin privileges
</ParamField>

### Body Parameters

<ParamField body="name" type="string" required>
  Category name (max 255 characters)
</ParamField>

<ParamField body="slug" type="string" optional>
  URL-friendly slug. Auto-generated from name if not provided (max 255 characters, must be unique)
</ParamField>

<ParamField body="description" type="string" optional>
  Category description
</ParamField>

<ParamField body="image_url" type="string" optional>
  URL to category image (must be valid URL)
</ParamField>

<ParamField body="is_active" type="boolean" optional>
  Whether the category is active (default: true)
</ParamField>

### Response

<ResponseField name="status" type="string">
  Response status ("success")
</ResponseField>

<ResponseField name="data" type="object">
  Created category object
</ResponseField>

<RequestExample>
```bash cURL
curl -X POST https://api.grancamerino.com/api/categories \
  -H "Authorization: Bearer YOUR_ADMIN_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "Kids",
    "description": "Children's jerseys and apparel",
    "image_url": "https://cdn.grancamerino.com/categories/kids.jpg",
    "is_active": true
  }'
```
</RequestExample>

<ResponseExample>
```json Response
{
  "status": "success",
  "data": {
    "id": "9d445a8c-8c3a-4d9d-bb4c-f0a6c2e8b125",
    "name": "Kids",
    "slug": "kids",
    "description": "Children's jerseys and apparel",
    "image_url": "https://cdn.grancamerino.com/categories/kids.jpg",
    "is_active": true,
    "created_at": "2024-03-03T14:25:00.000000Z",
    "updated_at": "2024-03-03T14:25:00.000000Z"
  }
}
```
</ResponseExample>
