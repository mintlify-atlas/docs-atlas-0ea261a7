## List All Countries

<api method="GET" url="https://api.grancamerino.com/api/countries" />

Retrieve all countries with their associated continent data.

### Response

<ResponseField name="status" type="string">
  Response status ("success")
</ResponseField>

<ResponseField name="data" type="array">
  Array of country objects
  
  <Expandable title="Country Object">
    <ResponseField name="id" type="string">
      UUID of the country
    </ResponseField>
    
    <ResponseField name="name" type="string">
      Country name (e.g., "Spain", "Brazil")
    </ResponseField>
    
    <ResponseField name="continent_id" type="string">
      UUID of the associated continent
    </ResponseField>
    
    <ResponseField name="image_url" type="string" optional>
      URL to country flag or emblem
    </ResponseField>
    
    <ResponseField name="created_at" type="string">
      ISO 8601 timestamp
    </ResponseField>
    
    <ResponseField name="updated_at" type="string">
      ISO 8601 timestamp
    </ResponseField>
    
    <ResponseField name="continent" type="object">
      Related continent object
      
      <Expandable title="Continent Object">
        <ResponseField name="id" type="string">
          Continent UUID
        </ResponseField>
        <ResponseField name="name" type="string">
          Continent name
        </ResponseField>
      </Expandable>
    </ResponseField>
  </Expandable>
</ResponseField>

## Create Country

<api method="POST" url="https://api.grancamerino.com/api/countries" auth="Bearer" />

<Note type="warning">
  **Admin Only** - Requires authentication with admin privileges
</Note>

Create a new country.

### Headers

<ParamField header="Authorization" type="string" required>
  Bearer token with admin privileges
</ParamField>

### Body Parameters

<ParamField body="name" type="string" required>
  Country name (max 255 characters)
</ParamField>

<ParamField body="continent_id" type="string" required>
  UUID of the continent where the country is located
</ParamField>

### Response

<ResponseField name="status" type="string">
  Response status ("success")
</ResponseField>

<ResponseField name="data" type="object">
  Created country object
</ResponseField>

<RequestExample>
```bash cURL
curl -X POST https://api.grancamerino.com/api/countries \
  -H "Authorization: Bearer YOUR_ADMIN_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "Portugal",
    "continent_id": "9d445a8c-8c3a-4d9d-bb4c-f0a6c2e8b501"
  }'
```
</RequestExample>

<ResponseExample>
```json Response
{
  "status": "success",
  "data": {
    "id": "9d445a8c-8c3a-4d9d-bb4c-f0a6c2e8b406",
    "name": "Portugal",
    "continent_id": "9d445a8c-8c3a-4d9d-bb4c-f0a6c2e8b501",
    "created_at": "2024-03-03T14:50:00.000000Z",
    "updated_at": "2024-03-03T14:50:00.000000Z"
  }
}
```
</ResponseExample>
