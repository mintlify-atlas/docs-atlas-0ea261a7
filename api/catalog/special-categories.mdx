## List All Special Categories

<api method="GET" url="https://api.grancamerino.com/api/special-categories" />

Retrieve all special categories. Special categories are used to group products by special collections, promotions, or featured items.

### Response

<ResponseField name="status" type="string">
  Response status ("success")
</ResponseField>

<ResponseField name="data" type="array">
  Array of special category objects
  
  <Expandable title="Special Category Object">
    <ResponseField name="id" type="string">
      UUID of the special category
    </ResponseField>
    
    <ResponseField name="name" type="string">
      Special category name (e.g., "New Arrivals", "Best Sellers", "Featured")
    </ResponseField>
    
    <ResponseField name="created_at" type="string">
      ISO 8601 timestamp
    </ResponseField>
    
    <ResponseField name="updated_at" type="string">
      ISO 8601 timestamp
    </ResponseField>
  </Expandable>
</ResponseField>

## Create Special Category

<api method="POST" url="https://api.grancamerino.com/api/special-categories" auth="Bearer" />

<Note type="warning">
  **Admin Only** - Requires authentication with admin privileges
</Note>

Create a new special category for organizing products.

### Headers

<ParamField header="Authorization" type="string" required>
  Bearer token with admin privileges
</ParamField>

### Body Parameters

<ParamField body="name" type="string" required>
  Special category name (max 255 characters)
</ParamField>

### Response

<ResponseField name="status" type="string">
  Response status ("success")
</ResponseField>

<ResponseField name="data" type="object">
  Created special category object
</ResponseField>

<RequestExample>
```bash cURL
curl -X POST https://api.grancamerino.com/api/special-categories \
  -H "Authorization: Bearer YOUR_ADMIN_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "Summer Collection 2024"
  }'
```
</RequestExample>

<ResponseExample>
```json Response
{
  "status": "success",
  "data": {
    "id": "9d445a8c-8c3a-4d9d-bb4c-f0a6c2e8b705",
    "name": "Summer Collection 2024",
    "created_at": "2024-03-03T15:20:00.000000Z",
    "updated_at": "2024-03-03T15:20:00.000000Z"
  }
}
```
</ResponseExample>
