## Overview

The Addresses API allows authenticated users to manage their delivery addresses. Users can create, update, delete, and set primary addresses for order checkout.

**Authentication:** All endpoints require Bearer token authentication.

---

## List Addresses

<api method="GET" path="/api/addresses" auth="required">
Get all addresses for the authenticated user.
</api>

### Response

<ResponseField name="status" type="string">
Response status ("success")
</ResponseField>

<ResponseField name="data" type="array">
Array of address objects
  <Expandable title="Address object">
    <ResponseField name="id" type="string">UUID of the address</ResponseField>
    <ResponseField name="user_id" type="string">UUID of the user</ResponseField>
    <ResponseField name="first_name" type="string">Recipient first name</ResponseField>
    <ResponseField name="last_name" type="string">Recipient last name</ResponseField>
    <ResponseField name="phone" type="string">Contact phone number</ResponseField>
    <ResponseField name="gender" type="string">Gender (male, female, other)</ResponseField>
    <ResponseField name="street" type="string">Street address line 1</ResponseField>
    <ResponseField name="street_line_2" type="string | null">Street address line 2 (optional)</ResponseField>
    <ResponseField name="city" type="string">City name</ResponseField>
    <ResponseField name="state" type="string">State or province</ResponseField>
    <ResponseField name="postal_code" type="string">Postal/ZIP code</ResponseField>
    <ResponseField name="country" type="string">Country name</ResponseField>
    <ResponseField name="is_primary" type="boolean">Whether this is the primary address</ResponseField>
    <ResponseField name="created_at" type="string">ISO 8601 timestamp</ResponseField>
    <ResponseField name="updated_at" type="string">ISO 8601 timestamp</ResponseField>
  </Expandable>
</ResponseField>

## Create Address

<api method="POST" path="/api/addresses" auth="required">
Create a new address for the authenticated user.
</api>

### Request Body

<ParamField name="first_name" type="string" required>
Recipient first name (max 255 characters)
</ParamField>

<ParamField name="last_name" type="string" required>
Recipient last name (max 255 characters)
</ParamField>

<ParamField name="phone" type="string" required>
Contact phone number (max 20 characters)
</ParamField>

<ParamField name="gender" type="string" required>
Gender. Must be one of: `male`, `female`, `other`
</ParamField>

<ParamField name="street" type="string" required>
Street address line 1 (max 255 characters)
</ParamField>

<ParamField name="street_line_2" type="string">
Street address line 2 - optional (max 255 characters)
</ParamField>

<ParamField name="city" type="string" required>
City name (max 255 characters)
</ParamField>

<ParamField name="state" type="string" required>
State or province (max 255 characters)
</ParamField>

<ParamField name="postal_code" type="string" required>
Postal or ZIP code (max 20 characters)
</ParamField>

<ParamField name="country" type="string" required>
Country name (max 255 characters)
</ParamField>

<ParamField name="is_primary" type="boolean" default="false">
Set as primary address. If true, other addresses will be set to non-primary automatically.
</ParamField>

### Response

<ResponseField name="status" type="string">
Response status ("success")
</ResponseField>

<ResponseField name="message" type="string">
Success message
</ResponseField>

<ResponseField name="data" type="object">
The created address object
</ResponseField>

### Example Request

```bash
curl -X POST https://api.grancamerino.com/api/addresses \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -H "Accept: application/json" \
  -d '{
    "first_name": "María",
    "last_name": "García",
    "phone": "+57 301 987 6543",
    "gender": "female",
    "street": "Carrera 7 # 50-20",
    "street_line_2": "Oficina 501",
    "city": "Medellín",
    "state": "Antioquia",
    "postal_code": "050001",
    "country": "Colombia",
    "is_primary": true
  }'
```

### Example Response

```json
{
  "status": "success",
  "message": "Address created successfully",
  "data": {
    "id": "9d4e8f12-3c4a-4b5e-8f9a-1b2c3d4e5f7c",
    "user_id": "9d4e8f12-3c4a-4b5e-8f9a-1b2c3d4e5f6b",
    "first_name": "María",
    "last_name": "García",
```
